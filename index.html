About Solus MS

Solus MS is a custom browser emulator launching interface designed by katsugachi on GitHub for playing games and launching your own games. It allows for a quick launch and effective gaming environment sandbox and custom game launching.

For Requests, Reviews And Discussion, Comment In Issues in https://github.com/Katsugachi/Solus-MS
